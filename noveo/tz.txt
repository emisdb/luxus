Цель этого задания – проверить ваши навыки разработки приложений с использованием PHP и Laravel. Пожалуйста, проявите свои способности к проектированию систем, написанию чистого и поддерживаемого кода.


1. Реализуйте систему управления задачами

Создайте базовую систему управления задачами. В системе должны быть пользователи и задачи. Каждый пользователь может иметь множество задач.


Логическая структура:

Пользователь: имя, email, пароль

Задача: Название, описание, дата завершения, статус (новая, в работе, завершенная), назначенный пользователь

Операции:

Регистрация/Аутентификация пользователей

Создание, чтение, обновление и удаление задач

Назначение задач пользователям

2. API

Создайте REST Api для выполнения операций, описанных выше. API должно быть защищено JWT (JSON Web Token) аутентификацией.


3. Разработайте систему уведомлений

Создайте систему которая будет отправлять уведомления пользователям о просроченных задачах. Уведомления должны отправляться по электронной почте и сохраняться в базе данных для истории уведомлений.


Пожалуйста, сохраните ваш код в репозитории Git и пришлите ссылку на ваш репозиторий вместе с вашими ответами. Оценивать мы будем структуру вашего кода, методологию тестирования, общую архитектуру приложения и ваш подход к решению проблем.
